---

    Windows (.bat files)

  - start_app.bat - Main script with full startup and shutdown logic
  - stop_app.bat - Quick shortcut for shutdown

  Linux/Mac (.sh file)

  - start_app.sh - Cross-platform shell script version

  Usage

  Windows:

  # Start all services
  start_app.bat

  # Stop all services
  start_app.bat stop
  # OR
  stop_app.bat

  Linux/Mac:

  # Start all services
  ./start_app.sh

  # Stop all services
  ./start_app.sh stop
